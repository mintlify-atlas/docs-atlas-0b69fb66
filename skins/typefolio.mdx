# Typefolio Skin

The **Typefolio** skin transforms your portfolio into a web-book reader with classic two-page spread layout and elegant typography.

## Overview

Typefolio presents works as a digital book with:
- Two-page spread layout (recto/verso pages)
- Caslon-style serif typography
- Page-by-page navigation
- Integrated PDF preview drawer
- Book-like reading experience

## Visual Style

- **Typography**: Caslon-style serif fonts with refined spacing
- **Layout**: Two-page spread (left/right pages)
- **Navigation**: Page counter and playback controls in footer
- **PDF Preview**: Slide-out drawer panel
- **Theme**: Light and dark modes with subtle transitions

## Generation Example

```bash
# Generate with Typefolio skin
praetorius generate --skin typefolio

# Using alias
praetorius generate --skin typography
```

## Configuration

Set Typefolio as default in `.prae/config.json`:

```json
{
  "ui": {
    "skin": "typefolio",
    "allowUrlOverride": true
  },
  "site": {
    "fullName": "Composer Name",
    "copyrightName": "© 2024 Composer Name"
  }
}
```

## Key Features

### Two-Page Spread
Works flow across facing pages like a physical book, with recto (right) and verso (left) page styles.

### Page Navigation
Footer displays current page and total page count (`0/5` format).

### PDF Preview Drawer
Slide-out panel (`#tf-preview`) shows PDF previews without disrupting the reading flow.

### Typography Controls
Optimized line length, leading, and hierarchy for long-form reading.

### Play Controls
Integrated audio playback button in footer (hidden when no audio available).

```html
  <main id="tf-book" class="tf-book">
    <div id="tf-flow" class="tf-flow"></div>
  </main>
  <aside id="tf-preview" class="tf-preview">
    <!-- PDF preview iframe -->
  </aside>
</div>

<footer id="tf-foot" class="tf-foot">
  <span class="tf-foot-pages">
    <span id="tf-page-now">0</span>/<span id="tf-page-total">0</span>
  </span>

```

## CSS Classes

- `.tf-book` - Main book container
- `.tf-flow` - Content flow container for pages
- `.tf-preview` - PDF preview drawer
- `.tf-foot-pages` - Page counter display
- `.tf-theme-toggle` - Theme switcher button

## Source Location

Implementation: `source/ui/skins/typefolio/`
- `template.html` - Book structure with spread layout
- `style.css` - Typography and page styles
- `main.js` - Page flow and preview logic
