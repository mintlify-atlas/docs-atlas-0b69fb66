# Vite Breeze Skin

The **Vite Breeze** skin is a sophisticated portfolio layout featuring a liquid glass aesthetic, split-pane design, and integrated PDF viewer.

## Overview

Vite Breeze provides a modern, professional portfolio experience with:
- Split layout: works list on the left, PDF viewer on the right
- Glass-morphism design with backdrop blur effects
- Integrated in-page PDF viewer (no popups)
- Smooth theme toggle with animated sun/moon icons
- Responsive navigation and footer

## Visual Style

- **Design**: Liquid glass with saturated blur and transparency
- **Layout**: Split shell (list + PDF pane)
- **Typography**: System UI fonts with clean hierarchy
- **Theme Toggle**: Animated switch with sun/moon icons
- **PDF Viewer**: Embedded iframe with close button

## Generation Example

```bash
# Generate with Vite Breeze skin
praetorius generate --skin vite-breeze

# Or configure as default
praetorius config set ui.skin vite-breeze
praetorius generate
```

## Configuration

Set Vite Breeze as default in `.prae/config.json`:

```json
{
  "ui": {
    "skin": "vite-breeze",
    "allowUrlOverride": true
  },
  "theme": "light",
  "site": {
    "fullName": "Your Name",
    "subtitle": "Selected works",
    "copyrightName": "Your Name",
    "links": [
      { "label": "Home", "href": "/", "external": false },
      { "label": "Contact", "href": "/contact", "external": false }
    ]
  }
}
```

## Key Features

### Split Layout
Works list (`#works-console`) on the left, PDF pane (`.vb-pdfpane`) on the right.

### Glass Morphism
Backdrop blur and saturation effects for modern depth and layering.

### Integrated PDF Viewer
In-page iframe viewer with header bar and close button—no external popups.

### Theme Toggle
Smooth animated toggle between light and dark modes with icon transitions.

### Responsive Header
Brand display, navigation links, theme toggle, and "Powered by Praetorius" badge.
