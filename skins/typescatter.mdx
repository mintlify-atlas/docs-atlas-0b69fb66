# TypeScatter Skin

The **TypeScatter** skin creates a type-only poster wall where work titles scatter across the screen and can be arranged into a grid layout.

## Overview

TypeScatter provides a unique, experimental interface with:
- Typography-only design (no images)
- Scattered poster layout that animates to grid
- Draggable work tiles
- Variable font sizes and rotations
- Minimalist, text-focused aesthetic
- Smooth scatter-to-grid transitions

## Visual Style

- **Typography**: Mixed sizes, weights, and rotations
- **Layout**: Random scatter → organized grid
- **Colors**: Monochrome or minimal color palette
- **Animation**: Smooth physics-based transitions
- **Interaction**: Draggable, re-arrangeable tiles

## Generation Example

```bash
# Generate with TypeScatter skin
praetorius generate --skin typescatter

# Using alias
praetorius generate --skin posterboard
```

## Configuration

Set TypeScatter as default in `.prae/config.json`:

```json
{
  "ui": {
    "skin": "typescatter",
    "allowUrlOverride": true
  },
  "typescatter": {
    "initialMode": "scatter",
    "enableDrag": true,
    "randomRotation": true,
    "fontSizeVariation": true
  }
}
```

## Key Features

### Scatter Mode
Work titles are randomly positioned across the viewport with varied sizes and rotations.

### Grid Mode
One-click transition to organized grid layout with uniform sizing.

### Drag & Drop
Users can drag individual tiles to custom positions (position persists in scatter mode).

### Typography Variation
Randomized font sizes (within bounds) and rotation angles for visual interest.

### Toggle Animation
Smooth spring-physics animation between scatter and grid modes.

### Click to Detail
Click any tile to view full work details.

## HTML Structure

```html
<div class="ts-container" data-mode="scatter">
  <header class="ts-header">
    <h1 class="ts-site-title">Portfolio</h1>
    <button id="ts-mode-toggle" class="ts-toggle" aria-pressed="false">
      <span class="ts-toggle-label">Grid</span>
    </button>
  </header>
  
  <main class="ts-stage">
    <div class="ts-tile" 
         data-work-id="1" 
         style="--x: 20%; --y: 30%; --rot: -5deg; --scale: 1.2;">
      <h2 class="ts-tile-title">Work Title</h2>
    </div>
    <!-- More tiles scattered... -->
  </main>
</div>
```

## CSS Classes

- `.ts-container` - Main container
- `.ts-stage` - Scatter/grid stage
- `.ts-tile` - Individual work tile
- `.ts-tile-title` - Work title text
- `.ts-toggle` - Scatter/grid mode toggle
- `.ts-container[data-mode="grid"]` - Grid mode state
- `.ts-container[data-mode="scatter"]` - Scatter mode state

## CSS Custom Properties

Each tile uses CSS custom properties for animation:

```css
.ts-tile {
  --x: 20%;           /* horizontal position */
  --y: 30%;           /* vertical position */
  --rot: -5deg;       /* rotation angle */
  --scale: 1.2;       /* size multiplier */
}
```

## Interaction Modes

### Scatter Mode (Default)
- Tiles randomly positioned
- Variable sizes and rotations
- Draggable to custom positions
- Organic, poster-wall aesthetic

### Grid Mode
- Organized rows and columns
- Uniform sizing and no rotation
- Maintains reading order
- Easier scanning and selection

## TypeScatter Options

### Initial Mode
```json
"typescatter": {
  "initialMode": "scatter"  // or "grid"
}
```

### Enable Dragging
```json
"typescatter": {
  "enableDrag": true
}
```

### Random Rotation
```json
"typescatter": {
  "randomRotation": true,
  "maxRotation": 15  // degrees
}
```

### Font Size Variation
```json
"typescatter": {
  "fontSizeVariation": true,
  "minFontSize": 1.0,  // rem
  "maxFontSize": 3.0   // rem
}
```

## Accessibility Considerations

<Warning>
Scatter mode can be disorienting for some users. Always provide:
- Clear toggle to grid mode
- Keyboard navigation support
- Screen reader-friendly DOM order
- Option to disable animations
</Warning>
