# Skins Overview

Praetorius provides multiple built-in **skins** (UI themes) to customize how your works are presented. Each skin is designed for specific use cases, from minimal console views to rich portfolio layouts.

## What are Skins?

Skins are complete UI templates that control:
- Layout and visual design
- Navigation patterns
- Media player controls
- PDF viewer integration
- Typography and spacing

## Configuring Your Skin

Set your default skin in `.prae/config.json`:

```json
{
  "ui": {
    "skin": "vite-breeze",
    "allowUrlOverride": true
  }
}
```

## Skin Aliases

Some skins have convenient aliases:
- `cards-tabs` → `dashboard`
- `docs-reader` → `docs`
- `kiosk` → `presentation`
- `typefolio` → `typography`
- `typescatter` → `posterboard`

## URL Override

When `allowUrlOverride` is enabled, preview different skins using query parameters:

```
http://localhost:3000?skin=typefolio
http://localhost:3000?skin=kiosk
```

## Source Code

All skin implementations are located in `source/ui/skins/` directory, each containing:
- `template.html` - HTML structure
- `style.css` - Skin-specific styles
- `main.js` - Interactive behavior
